# 3D Model Visualizer

This skill focuses on describing and generating high-quality 3D renders for products and architectural concepts.

## Instructions

1. Identify the object's material (Metal, Glass, Matte, Wood).
2. Set up the lighting environment (Studio, Natural, Neon).
3. Define the camera angle (Macro, Isometric, Wide-angle).
4. Generate the render using high-quality image generation tools.
5. Refine details like shadows and reflections for realism.

## Examples

- "Render a 3D isometric view of a sleek, white smart-home hub."
- "Create a photorealistic 3D visualization of a luxury watch with gold accents."

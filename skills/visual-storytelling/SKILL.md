# Visual Storytelling

This skill transforms complex data or abstract concepts into easy-to-understand visual narratives.

## Instructions

1. Simplify the complex idea into 3-5 core visual steps.
2. Choose a consistent illustrative style.
3. Create a series of visuals that follow a logical progression.
4. Use metaphors and analogies (e.g., a lighthouse for data monitoring).
5. Combine text and image for maximum clarity.

## Examples

- "Create an illustration series explaining how an AI neural network works."
- "Visualize the step-by-step process of a secure blockchain transaction."

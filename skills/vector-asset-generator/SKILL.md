# Vector Asset Generator

This skill enables the creation of scalable vector graphics for web and print.

## Instructions

1. Define the graphic's purpose (Background, Illustration, Divider).
2. Generate concepts using AI image tools or code directly for simple paths.
3. Convert bitmaps to SVG paths if necessary.
4. Optimize SVG code to remove metadata and reduce file size.
5. Ensure cross-browser compatibility for SVG filters and effects.

## Examples

- "Generate a minimalist SVG geometric background pattern for a website."
- "Create a scalable vector illustration of a robotic hand."

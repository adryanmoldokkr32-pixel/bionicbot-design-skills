# Kinetic Typography Orchestrator

This skill animates text to increase emotional impact and retention in video and web.

## Instructions
1. Analyze the spoken emphasis or the core message of the sentence.
2. Apply motion properties (Scale, Rotate, Blur) to key words.
3. Sync text transitions with the Marcus v2 audio peaks.
4. Use high-contrast colors (Gold/Neon) for emphasized terms.

# Minimalism Advisor

This skill focuses on the 'less is more' philosophy, removing friction and unnecessary elements to highlight what truly matters.

## Instructions

1. Analyze a design and identify 'clutter' (redundant borders, multiple font sizes, extra colors).
2. Suggest elements to remove or consolidate.
3. Increase whitespace (negative space) to improve focus.
4. Simplify the visual hierarchy to a maximum of 3 levels.
5. Optimize for cognitive ease and rapid comprehension.

## Examples

- "Simplify this hero section by removing 3 unnecessary elements."
- "Redesign this mobile menu using minimalist principles."

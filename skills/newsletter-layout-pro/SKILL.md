# Newsletter Layout Pro

This skill enables the creation of visually appealing and technically optimized email layouts for newsletters.

## Instructions

1. Use a single-column layout for maximum mobile compatibility.
2. Balance text-to-image ratio to avoid spam filters.
3. Place clear Call-to-Action (CTA) buttons in the top half (above the fold).
4. Optimize subject lines and preheader text for high open rates.
5. Ensure responsive design using nested tables or modern email frameworks.

- "Optimize this email template for better conversion on iPhone."

# Color Theory Strategist

This skill applies the principles of color psychology to create harmonious and effective palettes.

## Instructions

1. Select a base color that matches the brand's emotion.
2. Use color wheels to find complementary or analogous schemes.
3. Ensure high contrast for accessibility (WCAG compliance).
4. Define states (Success, Warning, Error) using color.
5. Provide Hex, RGB, and HSL codes.

## Examples

- "Create a calming color palette for a healthcare app."
- "Select vibrant colors for a gaming community portal."

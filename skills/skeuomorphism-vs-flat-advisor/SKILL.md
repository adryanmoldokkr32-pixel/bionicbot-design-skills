# Skeuomorphism vs Flat Advisor

This skill helps in choosing between realistic (skeuomorphic) and minimalist (flat/neumorphic) styles based on current trends and user needs.

## Instructions

1. Identify the project's target audience (General public vs Tech-savvy).
2. Recommend Flat design for speed and modern feel.
3. Recommend Skeuomorphism for tactile, intuitive interfaces (e.g., audio tools).
4. Suggest 'Glassmorphism' or 'Neumorphism' for a futuristic, hybrid look.
5. Provide CSS properties (shadows, blurs) to achieve the chosen style.

## Examples

- "Should I use flat design or skeuomorphism for a professional music production app?"
- "Give me the CSS for a modern glassmorphism card."

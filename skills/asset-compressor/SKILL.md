# Asset Compressor

This skill focuses on reducing the file size of images, videos, and fonts without sacrificing perceptible quality.

## Instructions

1. Choose the right format (WebP for images, MP4/WebM for video, WOFF2 for fonts).
2. Apply lossy or lossless compression techniques.
3. Remove unnecessary metadata (EXIF, GPS data).
4. Resize assets to their maximum display dimensions to avoid over-serving pixels.
5. Provide a 'Before vs After' size comparison.

## Examples

- "Compress this 5MB PNG into a WebP under 200KB."
- "Optimize this background video for a fast-loading landing page."

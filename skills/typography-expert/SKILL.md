# Typography Expert

This skill ensures that all text is legible, aesthetically pleasing, and aligned with the brand's tone.

## Instructions

1. Select font families based on the project's purpose.
2. Set font sizes, line heights, and letter spacing.
3. Create a clear typographic hierarchy (H1, H2, Body).
4. Ensure readability across different device sizes.
5. Recommend pairings from Google Fonts or Adobe Fonts.

## Examples

- "Suggest a font pairing for a high-end luxury fashion website."
- "Optimize the typography of my blog for better readability."

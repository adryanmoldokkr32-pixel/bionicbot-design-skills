# Design System Manager

This skill ensures that every element of a project follows a unified set of rules and reusable components.

## Instructions

1. Inventory existing UI elements (Buttons, Inputs, Modals).
2. Define naming conventions (Design Tokens) for colors and spacing.
3. Create documentation for reusable component states.
4. Audit new designs against the established system.
5. Suggest updates to the system as the project evolves.

## Examples

- "Create a design system documentation for my 'Adrianfinance26' project."
- "Check if this new button design matches our current style guide."

# Accessibility Auditor (WCAG)

This skill ensures that digital products are accessible to everyone, including people with disabilities, by following Web Content Accessibility Guidelines.

## Instructions

1. Check color contrast ratios (text vs background).
2. Ensure interactive elements have sufficient tap targets for mobile.
3. Verify that all images have descriptive Alt text.
4. Check for logical heading hierarchy and keyboard navigation support.
5. Provide suggestions for making forms more screen-reader friendly.

## Examples

- "Audit the accessibility of this landing page mockup."
- "Suggest 3 changes to make this dashboard more inclusive."

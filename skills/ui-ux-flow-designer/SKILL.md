# UI/UX Flow Designer

This skill focuses on the logical structure and interactive experience of a digital product.

## Instructions

1. Identify user goals and pain points.
2. Draft a site map or app architecture.
3. Design wireframes for key screens.
4. Define interactive transitions and feedback loops.
5. Optimize the flow for minimum friction and maximum conversion.

## Examples

- "Design the onboarding flow for a mobile meditation app."
- "Create a wireframe for a real-estate dashboard."

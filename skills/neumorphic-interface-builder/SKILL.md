# Neumorphic Interface Builder

This skill focuses on creating 'Soft UI' that mimics real-world physical objects through complex CSS box-shadows.

## Instructions
1. Select a base color with high luminance.
2. Define light-source direction (e.g., Top-Left).
3. Apply two shadows: one light (highlight) and one dark (depth).
4. Ensure accessibility by adding subtle borders for high-contrast needs.

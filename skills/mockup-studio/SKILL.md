# Mockup Studio

This skill transforms flat designs into professional presentations by placing them in realistic device frames (iPhone, Laptop) or scenes.

## Instructions

1. Select the appropriate device based on the design (e.g., iPhone 15 for a mobile app).
2. Generate a high-quality scene using image generation tools.
3. Ensure the design is correctly aligned and perspective-matched to the screen.
4. Use realistic lighting, shadows, and reflections.
5. Provide a set of mockups for a complete professional portfolio.

## Examples

- "Create a mockup of my meditation app on an iPhone 15 in a natural outdoor setting."
- "Showcase this dashboard design on a high-end laptop mockup."

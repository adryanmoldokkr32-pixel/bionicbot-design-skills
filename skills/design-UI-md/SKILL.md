# Design UI Skill

This skill enables the agent to design and implement professional user interfaces using modern CSS frameworks and design principles.

## Instructions
1. Analyze user design requirements.
2. Draft a layout using Tailwind CSS.
3. Implement responsive design for all screen sizes.
4. Apply consistent color theory and typography.
5. Optimize for mobile devices (iPhone 15).

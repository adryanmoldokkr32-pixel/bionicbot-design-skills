# Brand Identity Architect

This skill enables bionicbot to design logos, select typography, and define color palettes to create a unique and professional brand identity.

## Instructions

1. Analyze user's industry and target audience.
2. Suggest 3 distinct brand directions (e.g., Bold, Minimalist, Classic).
3. Generate logo concepts using image generation tools.
4. Define primary, secondary, and accent colors.
5. Select font pairings for headers and body text.

## Examples

- "Create a brand identity for a new eco-friendly coffee shop."
- "Design a logo and color palette for a tech startup in the AI space."

# Iconography Master

This skill focuses on creating a unified visual language through icons.

## Instructions

1. Define the icon style (Line, Filled, Duotone).
2. Create a set of core icons (Home, Settings, Profile, etc.).
3. Ensure consistent stroke weight and corner radius.
4. Optimize icons for different sizes (16px to 64px).
5. Export in SVG or PNG format.

## Examples

- "Generate a set of 10 minimalist outline icons for a fintech app."
- "Design a custom icon for a 'Save to Favorites' feature."
